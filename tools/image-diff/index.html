How this screenshot diff tool works

This tool compares two images pixel by pixel using your browser's Canvas API. Nothing is uploaded to any server — your images stay on your machine.

  • Highlight mode overlays changed pixels in red on the original image
  • Side by side shows both images next to each other for manual comparison
  • Diff only shows just the changed pixels on a dark background

Common uses: comparing website screenshots before and after a deploy, checking UI regressions, verifying design changes, auditing competitor page updates.
